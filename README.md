# Profile
My personal portfolio website showcasing my projects, skills, and live coding activity from LeetCode &amp; GitHub. Built with HTML, CSS, and JavaScript.

# 👨‍💻 Gagan B - Personal Portfolio

Welcome to my personal portfolio website! This site serves as a central hub to showcase my academic background, technical skills, projects, and real-time coding achievements.

🔗 **Live Demo:** [Click here to view my portfolio](https://profile-delta-sable.vercel.app/)

## 🚀 Features

* **Live Coding Heatmaps:** Dynamically fetches and displays my submission history from **LeetCode** and **GitHub** using JavaScript APIs.
* **Responsive Design:** Fully responsive layout that works on desktop and mobile (with a collapsing sidebar animation).
* **Project Showcase:** Highlights my key projects like *AgriSmart*, *Personal Fitness Tracker*, and *Smart-Education*.
* **Interactive UI:** Includes scroll-reveal animations and a typewriter effect for a modern user experience.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **APIs Used:**
    * LeetCode Stats API (for submission calendar & streak)
    * GitHub Contributions API (for contribution graph)

## 📂 Project Structure

* `index.html` - The main structure of the portfolio.
* `ProfileStyle.css` - Custom styling for animations, dark mode theme, and responsiveness.
* `ProfileScript.js` - Handles API fetching, streak calculations, and scroll animations.
